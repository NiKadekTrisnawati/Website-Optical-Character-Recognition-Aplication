What Is OCR?

Optical Character Recognition, commonly abbreviated (OCR) is a software that converts text in a format image or picture files into text format that can be read and edited by computer applications (for example: Notepad and Microsoft Word). OCR allows doing Text conversion from physical documents, such as books, newspapers, or scanned documents, to a digital format that can be edited. This makes it easier to edit, index, and reuse information. This simple OCR application uses the Python programming language with the PyQt5, tkinter, PIL (Python Imaging Library), OpenCV (cv2), numpy, pyperclip, and pytesseract libraries with an application size of 206 MB

Tutorial

Download the application by clicking on the download section in the navigation or scrolling down to find it

In the image below is a display of the OCR application, when you want to carry out the OCR process press the "snip and copy" button

After the ''snip and copy'' button is pressed the cursor will change to a plus sign, press and drag the cursor on the desired object, release the cursor then the object will be selected

After that, You can directly paste it in the desired place or you can also edit the selected object if there are words that are not appropriate before pasting.
